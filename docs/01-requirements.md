# 项目需求文档

## 产品概述
中级财务管理考试智能学习平台，支持网页端（PC + 手机浏览器）和微信小程序（后期），通过AI解析PDF教材，生成知识框架和个性化题库。

## 目标用户
备考中级财务管理考试的考生，个人使用为主。

## 功能需求

### F1 首页仪表盘
- 2026考季备考指南重点展示
- 2026教材变动说明要点展示
- 个人学习进度概览（学习天数、完成章节、正确率、薄弱点数量）

### F2 知识框架（思维导图）
- 上传PDF后AI自动拆解：科目 → 章 → 节 → 知识点
- 树形结构展示，支持展开/收起
- 知识点染色：已掌握(绿) / 薄弱(黄) / 未学(红)
- 点击查看详细内容

### F3 刷题模块
- 按章节刷题（顺序/随机）
- 智能出题（针对薄弱点）
- 模拟考试（随机组卷，限时）
- 逐题展示 → 作答 → 立即显示答案+解析
- 题型：单选、多选、判断、计算题

### F4 智能分析与推荐
- 记录每道题作答情况
- 知识点掌握度分析
- 薄弱点自动识别
- 针对性题库滚动推送

### F5 搜索功能
- 关键词搜索知识点
- 搜索题目

### F6 后台管理
- PDF上传与解析触发
- AI解析结果校对编辑
- 题库手动增删改
- 知识点框架手动调整

## 非功能需求
- 响应式设计，适配PC和手机
- 淡蓝色主色调，简洁直观
- 数据云端存储，多设备访问
- 低成本方案，优先使用免费服务
