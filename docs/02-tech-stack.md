# 技术栈文档

## 前端
- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件库**: shadcn/ui (基于 Radix UI)
- **状态管理**: React Context + SWR (数据请求)

## 后端
- **API**: Next.js API Routes / Route Handlers
- **ORM**: Prisma
- **认证**: Supabase Auth (初期可选单用户模式)

## 数据库
- **服务**: Supabase (PostgreSQL)
- **文件存储**: Supabase Storage

## AI服务
- **PDF文本提取**: pdf-parse (pdf.js)
- **内容结构化**: DeepSeek API / OpenAI API
- **用途**: 章节拆解、知识点提取、题目识别

## 部署
- **前端+API**: Vercel (免费计划)
- **数据库**: Supabase (免费计划)

## 开发工具
- **包管理**: npm
- **代码格式化**: Prettier (可选)
- **版本控制**: Git
